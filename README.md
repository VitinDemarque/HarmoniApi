## Instalação
  1 - Clone o Repositório do GitHub:
  ```bash
    git clone https://github.com/VitinDemarque/HarmoniApi
  ```

  2 - Instale as Dependências:
  ```bash
    npm install
  ```

  3 - Execute a Aplicação:
  ```bash
    npm start
  ```
  A aplicação estará disponível em http://localhost:3000
